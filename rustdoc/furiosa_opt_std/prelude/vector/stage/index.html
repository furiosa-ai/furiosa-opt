Vector Engine stage markers and state.

This module provides stage markers and state tracking for the Vector Engine pipeline. The actual tensor types are in the tensor module.

§Stage Order

Tag → Logic → Fxp → FxpToFp → Narrow → Fp → IntraSliceReduce → FpDiv → Widen → FpToFxp → Clip → Output

§Modules

  • markers: Stage marker types and transition traits
  • state: VE state tracking

Stashable
Trait for stages that support stash operation. Only specific stages can stash current data (based on hardware ALU availability). Supported: Tag, Logic, Fxp, Narrow, Fp, FpDiv, Clip NOT supported: FxpToFp, IntraSliceReduce, Widen, FpToFxp, Output
VeTensorContext
Marker trait for VE tensor context state. Used to distinguish tensor contexts and restrict available operations. there are three Contexts: Standalone/Group/Zipped. depending on the TensorContext, commit/stash/filter operations may be allowed or disallowed.

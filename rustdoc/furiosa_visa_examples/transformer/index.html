Qwen 2.5 0.5B composable transformer kernels (4PE, W16A16KV16, bf16).

Each module maps to a composable kernel stage:

ModuleLayer range
embeddingembedding..transformer_0:qkv_projection
attentiontransformer_N:attention (prefill, S=1024)
decodertransformer_N:output_projection..transformer_N+1:qkv
headtransformer_-1:output_projection..output (+ LM head)

Shared building blocks (O-proj, MLP, residual+norm, RoPE) live in common/.

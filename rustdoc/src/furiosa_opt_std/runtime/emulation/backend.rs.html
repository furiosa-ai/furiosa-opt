9/// Emulation backend: host-side buffer emulator using `BufRawTensor` storage.
10///
11/// Holds a native `Vec<D>` host staging buffer per tensor, the same storage shape used by `Npu`,
12/// but without an actual device behind it. Unlike `Simulation` (which interprets every operation

14/// where buffer-level semantics will eventually be implemented. Today the value-producing
